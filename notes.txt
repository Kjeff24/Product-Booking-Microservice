Installing vault:
    vault server --dev --dev-root-token-id="00000000-0000-0000-0000-000000000000"
Set vault address in environment variable:
    variable name: VAULT_ADDR
    variable path: http://127.0.0.1:8200
Encrypting a file using vault:
    vault kv put secret/<file-name> @<file-name-with-path>
    Example:
  vault kv get secret/booking-service
